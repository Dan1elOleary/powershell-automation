# PowerShell Automation

This repository contains PowerShell scripts created for IT support, Windows administration, endpoint troubleshooting, and managed services workflows.

The purpose of this repository is to showcase practical PowerShell automation that can help collect system information, speed up troubleshooting, document workstation health, and support common help desk or MSP tasks.

## Scripts Included

| Script | Purpose |
|---|---|
| workstation-health-check.ps1 | Collects workstation health information such as hostname, uptime, OS version, disk space, network details, and connectivity status. |
| disk-space-report.ps1 | Checks disk usage and warns when free space is low. |
| network-info-check.ps1 | Displays IP address, DNS servers, gateway, and adapter information. |
| bitlocker-status-check.ps1 | Checks BitLocker encryption status on Windows drives. |
| installed-software-inventory.ps1 | Lists installed software for inventory and troubleshooting. |
| user-profile-cleanup-check.ps1 | Reviews local user profiles and identifies older profiles that may need review. |
